## Project Structure

This is a **full-stack** application with React frontend and Express backend.

### Tech Stack
**Frontend:**
- ⚛️ React 18 + Vite
- 🎨 Tailwind CSS
- 🧭 React Router DOM
- 📡 Axios

**Backend:**
- 🚀 Express.js
- 🍃 MongoDB (Mongoose)
- 🔧 Node.js (ES6 Modules)

### Getting Started

**Start the backend:**
```bash
cd backend
npm run dev
```

**Start the frontend (in a new terminal):**
```bash
cd frontend
npm run dev
```

- Frontend: http://localhost:5173
- Backend: http://localhost:5000
